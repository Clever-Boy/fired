F.I.R.E.D.
==========


Description
-----------

F.I.R.E.D. is a 2D cross-platform games with elements of Action, RPG and Adventure.


Controls
--------

`A` - move left
`D` - move right
`SPACE` - jump
`S` - jumpdown
`LEFT MOUSE` - shot
`Q` - switch weapon
`E` - interact
`C` - character window
`I` - inventory window
`ESC` - main menu
